# A³ Core Usermanual
Scalable "set and forget" soundprocessor

## audio io-layout
It is free configurable but comes with a preset layout

### inputs
- 1-2 = Stereo 1
- 3-4 = Stereo 2
- 5-6 = Stereo 3
- 7-8 = Stereo 4
- 9-24 = Tape-In

### outputs
- 1-2 = A³ Mix
- 3 = Subwoofer
- 4-n = Speaker

## Service
- Clean fan-filters regulary
